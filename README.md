# Portfolio GitHub - Daisan Tofani

Website portofolio elegan berbasis HTML, CSS, dan JavaScript.

## Isi File
- `index.html` : Struktur halaman portofolio.
- `style.css` : Desain tampilan elegan dan responsif.
- `app.js` : Script menu, animasi, dan integrasi GitHub API.
- `app.jss` : Salinan dari `app.js` untuk menyesuaikan permintaan nama file.

## Cara Menghubungkan ke GitHub
1. Buka file `index.html`.
2. Cari bagian:
   ```html
   <body data-github-user="GITHUB_USERNAME">
   ```
3. Ganti `GITHUB_USERNAME` dengan username GitHub kamu.
   Contoh:
   ```html
   <body data-github-user="daisantofani">
   ```
4. Simpan file.
5. Buka `index.html` di browser.

## Deploy ke GitHub Pages
1. Buat repository baru di GitHub.
2. Upload semua file ke repository.
3. Masuk ke Settings > Pages.
4. Pada bagian Branch, pilih `main` dan folder `/root`.
5. Klik Save.
6. Website akan aktif melalui link GitHub Pages.
